# recipes
Simple CRUD restful api with [Express](http://expressjs.com/) and [MongoDB](https://www.mongodb.com/)

## Getting Started

### Prerequisites

* Windows
* Node.js
* MongoDB

### Installing

```sh
git clone  https://github.com/okunoyuki/recipes.git
cd recipes
npm install

# This command starts MongoDB server
npm run db_start

# On another window
npm start
```
